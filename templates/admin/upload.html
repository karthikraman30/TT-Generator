File Format Guide

1. Slots File: The standard university slots Excel file. Columns grouped by slots (Slot-1 through Slot-8, Slot-Free). Each row maps a batch to its courses, L-T-P-C, course type, and faculty abbreviation.

2. Rooms File: Excel/CSV with columns: Room Name, Capacity, Building (optional), Type (optional).

3. Section Strengths: Excel/CSV with 2 columns: Section / Batch Label and Student Count. Example:

Label Count Sec A180 Sec B180 CS-Only80 EVD40 MnC60 MTech ICT-ML30

A reference file section_strengths.xlsx has been generated in your project folder. Fill in the counts and upload it.

4. Faculty Mapping: CSV with columns: Abbreviation, Full Name, Email (optional).

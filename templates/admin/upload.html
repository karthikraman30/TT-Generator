File Format Guide

1. Slots File: The standard university slots Excel file. Columns grouped by slots (Slot-1 through Slot-8, Slot-Free). Each row maps a batch to its courses, L-T-P-C, course type, and faculty abbreviation.

2. Rooms File: Excel/CSV with columns: Room Name, Capacity, Building (optional), Type (optional).

3. Course Strengths Recommended: Excel/CSV with 2 columns: Course Code and Required Seats. Directly sets room capacity per course.

Course Code Required Seats IT301180 SC20580 MA203360

4. Section Strengths: Excel/CSV with 2 columns: Section / Batch Label and Student Count. Uses pattern matching (Sec A, Sec B, CS-Only, etc.).

5. Faculty Mapping: Excel/CSV with columns: Full Name, Abbreviation, Email (optional). Maps names to abbreviations already imported from Slots file.
